# Credit Card Fraud Detection

## Project Overview
This project aims to detect fraudulent credit card transactions using Machine Learning techniques and visualize the results using Power BI. The model analyzes transaction data to identify suspicious activities and help prevent financial fraud.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Fraud detection using Machine Learning
- Performance evaluation of the model
- Interactive Power BI dashboard
- Fraud transaction analysis and visualization

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Power BI

## How to Run
1. Clone the repository.
2. Install the required Python libraries.

4. Run all cells.
5. Open `fraud_detection.pbix` in Power BI Desktop to view the dashboard.
